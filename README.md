# Newshub
基于React构建的个人新闻推荐应用

## 功能

  - Front-End：新闻收藏/隐藏/赞同/反对，视图切换，排序组合，分页缓存，记录搜索历史，中英文切换，消息通知

  - Web Server：基于JWT的登录注册

  - Back-End Service：RPC Server，新闻监测/爬取/去重，点击偏好模型，新闻主题模型
